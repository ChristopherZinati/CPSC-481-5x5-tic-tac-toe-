**_required libraries to run:_**

- tkinter

**_to run_**

1. in the terminal, change the directory to CPSC-481-5x5-tic-tac-toe- :

```sh
cd your/path/to/CPSC-481-5x5-tic-tac-toe-
```

2. run the script:

```sh
python main.py
```
